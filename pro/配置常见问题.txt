1.权限问题：
    打开页面报错信息提示打开文件错误： 检查项目文件权限 修改为777

2.404 nginx 404 Not Found：
    首页可以打开，子目录报错404： 网站--> 配置文件(nginx配置)增加：
    location ~ .*\.(gif|jpg|jpeg|png|bmp|swf)$
        {
            expires      30d;
            error_log off;
            access_log /dev/null;
        }

    在此项目后增加以下配置：

    location / {
        	if ( !-e $request_filename) {
                    rewrite ^/(.*)$ /index.php/$1 last;
                    break;
            }
        }
